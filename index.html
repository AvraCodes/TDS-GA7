# compute compound growth rate (CAGR)
def cagr(start_value, end_value, years):
    return (end_value / start_value) ** (1.0 / years) - 1

Financial Formula

Net Present Value (NPV):

$$\mathrm{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1 + r)^t}$$

Where $CF_t$ is cash flow at time $t$ and $r$ is discount rate.

Use this slide when discussing valuation or capital allocation decisions. Optionally show an example with numbers.
